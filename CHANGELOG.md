# Changelog

Todos los cambios importantes de este proyecto se documentaran en este archivo.

## [1.0.0] - 2026-06-08

### Agregado

- Búsqueda de ciudades mediante Open-Meteo Geocoding API.
- Clima actual.
- Pronóstico por horas.
- Pronóstico extendido de 7 días.
- Información detallada del viento.
- Sensación térmica.
- Humedad relativa.
- Presión atmosférica.
- Probabilidad de lluvia.
- Horarios de amanecer y anochecer.
- Diseño responsive.
- Soporte para iconos meteorológicos personalizados.

### Refactorizado

- Reorganización del CSS en módulos.
- Separación de la lógica en:
  - API
  - UI
  - Utilidades
