# CloudyApp

Aplicación web meteorológica desarrollada con JavaScript, HTML y CSS.

Permite consultar el clima actual y el pronóstico de cualquier ciudad utilizando la API de Open-Meteo.

## Versión

**1.0.0**

## Características

- Consulta meteorológica por ciudad.
- Clima actual en tiempo real.
- Temperatura actual.
- Sensación térmica.
- Humedad relativa.
- Presión atmosférica.
- Probabilidad de lluvia.
- Horarios de amanecer y anochecer.
- Pronóstico por horas.
- Pronóstico extendido de 7 días.
- Información detallada del viento.
- Iconos meteorológicos personalizados.
- Diseño responsive para dispositivos móviles y escritorio.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (ES6 Modules)
- Open-Meteo API
- Open-Meteo Geocoding API
